# 终极指南：在私有 PC 上彻底清洗并整理 Google 照片 (去重、合并相册)

由于极度涉及个人隐私（尤其是包含 NSFW 内容的照片），将 22GB 的 Google 照片扔到第三方服务器（如 GitHub Actions）或是任由商业去重软件扫描都是**极其危险**的。

这里提供**100% 绝对本地化、断网可执行的“防呆”操作指南**。不仅能清洗重复文件，还能自动将类似于 `相册A` 和 `相册A (1)` 的同名冗余相册合并到一个文件夹下。

## 📍 第一部分：从 Google 官方把照片全部安全运回本地 (Takeout)

只有将“数据控制权”掌握在自己硬盘里，我们才能随心所欲地整理。

1. **登录大本营**：在个人 PC 的浏览器中打开 [Google Takeout (导出数据)](https://takeout.google.com/)。
2. **选择导出项**：
   - 点击界面上的 **“取消全选”**（千万别导出所有服务，那是几百 GB 的灾难）。
   - 向下滚动找到 **“Google 照片”**，在后面的复选框打上勾。
   - *(可选)* 您可以点击下方的“已包含所有照片相册”，只勾选想要整理的特定相册，但我推荐全选。
3. **设置打包格式**：
   - 滑到最下面，点击 **“下一步”**。
   - 文件类型：选择 **.zip**。
   - 文件大小上限：**强烈推荐选择 10GB 甚至 50GB**（如果你的网络够稳定）。选 2GB 的话，你的 22GB 数据会被硬生生切成十几个压缩包，解压起来极其痛苦且容易漏掉。
4. **耐心等待**：Google 会在后台打包，等几个小时后，你会收到一封包含了下载链接的邮件。
5. **解压安放**：
   - 在个人电脑上准备一块有 **60GB+ 空闲空间** 的硬盘分区（SSD 最佳）。
   - 将下载回来的 ZIP 压缩包集中解压到一个专属目录，例如：`D:\GoogleTakeout\`
   - 点开观察，你会看到路径通常是类似这样的：`D:\GoogleTakeout\Takeout\Google 照片\`，里面躺着一年年的日期文件夹和你那些同名的相册（比如 *机车之旅*, *机车之旅 (1)*）。

## 📍 第二部分：部署并运行我们的“超级清洗脚本”

我们为你量身定制好了极速去重聚合的 Python 脚本，使用底层 MD5 字节校验，绝不冤枉一张原创照片，也绝不放过一张重复图片。

### 1. 环境准备 (如果个人 PC 上没装 Python)
- 去 [Python 官网](https://www.python.org/downloads/) 下载最新的 Windows 安装包。
- **重中之重**：安装向导的第一步，务必勾选底部那个极不显眼的 **"Add Python to PATH"** ！然后点击 `Install Now` 即可。

### 2. 获取清洗脚本
- 我已经把脚本写在了你的工作项目的这个位置：`e:\work\xai_public_staging\tools\google_photos_deduper.py`
- 请通过微信、U盘或者随手发个邮件等方式，把这个 `google_photos_deduper.py` 脚本文件传送到你的私人 PC 上，丢到桌面上或者某个好找的地方。

### 3. 配置脚本参数 (打开它！)
在你个人的 PC 上，右键点击 `google_photos_deduper.py` -> **使用记事本打开 (Edit with Notepad/IDLE)**。在代码的最上方，找到配置区并根据你的真实路径进行修改：

```python
# ==========================================
# 🛑 核心配置区域
# ==========================================

# 1. 指向你刚才解压完的 Google 相册根目录
SOURCE_DIR = r"D:\GoogleTakeout\Takeout\Google 照片"

# 2. 脚本整理后，最终纯净的相册库你要放到哪里？(任意新建个空文件夹就行)
DEST_DIR = r"D:\GooglePhotos_Cleaned"

# 3. 鉴定出来的重复垃圾要放到哪里？(给自己留退路，别直接删)
DUPLICATES_DIR = r"D:\GooglePhotos_Duplicates"

# 4. 最神奇的开关：安全演习模式
DRY_RUN = True
```

### 4. 万无一失的执行步骤

**第一波：安全演习 (Dry Run)**
在当前存放脚本的文件夹，按住键盘的 `Shift` 键，然后在文件夹空白处 `点右键`，选择 **“在此处打开 PowerShell 窗口”** (或者终端界面)。
输入下面这行命令敲回车：
```bash
python google_photos_deduper.py
```
*你会看到屏幕飞速滚动日志。由于现在 `DRY_RUN = True`，你的任何文件都不会被真实移动。仔细看看最后那华丽的“结算报告”，看看识别出了多少重复项，是否符合你的预期。*

**第二波：破釜沉舟 (真刀真枪清理)**
如果你对演习结果感到非常踏实。
1. 回到记事本，将代码里的 `DRY_RUN = True` 修改为 `DRY_RUN = False`。按 `Ctrl+S` 保存。
2. 回到黑框框命令台，按一下键盘的 `向上箭头 ↑`，重新执行一遍 `python google_photos_deduper.py`。
3. 等待这段奇妙的旅程结束。
